# Swift-E-Commerce

1. null মানে কোন ডাটা রাখা হয় নাই। undefined মানে আগে assinged করা হয়েছে কিন্তু এখনও value দেওয়া হয় নাই।
2. forEach শুধু values প্রিন্ট করে কিন্তু কোন কিছু return করে না, অন্য দিকে map নতুন করে array return করে।
3. == মানে সমান হতেও পারে আবার নাও হতে পারে, কিন্তু === মানে উভয় পক্ষ সমান হবেই।
4. sychronous ভাবে ডাটা load করে, সহজে error catch করা যায়।
5. scope হলো কোথায় থেকে variables accessed করা যাবে, global scope এ যেকোনো জায়গা থেকে variable পাওয়া যায়, function scope হলো শুধু মাত্র function এর ভিতর থেকে পাওয়া যাবে, block scope হলো শুধু মাত্র {} এর মাঝে বিদ্যমান variable.
